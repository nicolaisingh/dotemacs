# Emacs Lisp Expert Skill

## Core Competencies

### Emacs Configuration & Customization
- `.emacs` and `init.el` file structure and organization
- Package management with `package.el`, `use-package`, and `straight.el`
- Theme and UI customization
- Keybinding configuration and keymaps
- Mode-line and frame configuration
- Buffer and window management

### Emacs Lisp Language Expertise
- Basic syntax: atoms, lists, symbols, strings, numbers
- Special forms: `defun`, `defvar`, `defcustom`, `setq`, `let`, `let*`
- Control structures: `if`, `cond`, `when`, `unless`, `while`, `dolist`, `dotimes`
- Function definition and lambda expressions
- Macros: `defmacro`, macro expansion, common macro patterns
- Error handling: `condition-case`, `ignore-errors`, `error`, `signal`
- Debugging: `debug`, `edebug`, `debug-on-entry`, `debug-on-error`

### Common Emacs APIs
- Buffer manipulation: `with-current-buffer`, `save-excursion`, `save-restriction`
- Text manipulation: point, mark, region functions
- File and directory operations
- Process management and subprocesses
- Regular expressions and string matching
- Time and timer functions
- Hooks and advice (`add-hook`, `advice-add`)

### Development Workflow
- Interactive development with `M-x eval-buffer`, `M-x eval-region`, `C-M-x`
- Using the `*scratch*` buffer for experimentation
- Byte compilation and performance optimization
- Writing tests with `ert` (Emacs Lisp Regression Testing)
- Documentation strings and `C-h f`, `C-h v` integration
- Version control integration with VC and Magit

### Advanced Topics
- Writing major and minor modes
- Creating custom faces and text properties
- Implementing interactive commands
- Working with overlays and text properties
- Asynchronous programming with timers and processes
- Extending Emacs with external libraries
- Performance profiling and optimization

### Debugging Techniques
- Using `M-x toggle-debug-on-error`
- The `*Backtrace*` buffer and stack traces
- `M-x edebug-defun` for step-by-step debugging
- Inspecting variables with `M-x describe-variable`
- Using `M-x apropos` to find functions and variables
- The `*Messages*` buffer for logging

### Best Practices
- Writing idiomatic Emacs Lisp code
- Proper use of lexical vs dynamic binding
- Managing side effects and purity
- Error handling and user feedback
- Performance considerations for large buffers
- Writing maintainable and documented code
- Following Emacs coding conventions
